# HomeAutonom — iAiFy Enterprise

Smart home-automation och IoT-infrastruktur: Homey Pro, Home Assistant och Raspberry Pi-baserad natsverkshantering.

## Flaggskeppsrepos

- [homey-smart-home](https://github.com/HomeAutonom/homey-smart-home) — Full-stack smart home med 132 backend- och 67 dashboard-moduler (v3.3.0)
- [homey-automation](https://github.com/HomeAutonom/homey-automation) — Autonom automation och realtidsdashboard med Express 5 och Socket.IO (v3.261.0, 16500+ tester)
- [home-auto](https://github.com/HomeAutonom/home-auto) — Smart home-dashboard: Next.js 16, React 19, Supabase
- [home-network](https://github.com/HomeAutonom/home-network) — Raspberry Pi Docker-server med Pi-hole, Home Assistant och Grafana (69 branches, aktiv dev)
